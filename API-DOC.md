### 8. Get All Flight Data Between Bounds


Returns information of all detected flights in a rectangle bounded by two coordinates: **CoordA** e **CoordB.**

## Parameters
***{lat_a}*** - Latitude of **CoordA**

***{long_a}*** - Longitude of **CoordA**

***{lat_b}*** - Latitude of **CoordB**

***{long_b}*** - Longitude of **CoordB**


***Endpoint:***

```bash
Method: GET
Type: RAW
URL: https://piavia.serveo.net/adsb/{lat_a}/{long_a}/{lat_b}/{long_b}
```



***Responses:***
